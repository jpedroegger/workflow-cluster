Detailed Description

The CameraWidget class is a custom QWidget that displays a live camera feed.

This widget encapsulates a QCamera, QMediaCaptureSession, and QVideoWidget to provide a simple interface for displaying a camera feed in a Qt application. It supports customization of position and size.
