\doxysubsection{Detailed Description}
The \doxylink{classMainWindow}{Main\+Window} class represents the main window of the application. 

This class inherits from QMain\+Window and serves as the primary interface for the application. It manages the user interface components defined in the Ui\+::\+Main\+Window namespace.
