\doxysubsection{Detailed Description}
The \doxylink{classRosNode}{Ros\+Node} class represents a ROS 2 node for managing vehicle data. 

This class inherits from rclcpp\+::\+Node and subscribes to topics such as battery level and speed. It also includes methods to retrieve the latest values of these parameters.
