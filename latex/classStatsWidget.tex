\doxysubsection{Detailed Description}
The \doxylink{classStatsWidget}{Stats\+Widget} class represents a custom widget for displaying statistics. 

This widget provides a graphical representation of four key statistics\+:
\begin{DoxyItemize}
\item Distance Traveled
\item Average Speed
\item Battery Consumed
\item Obstacles Detected 
\end{DoxyItemize}
