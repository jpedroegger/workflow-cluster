\doxysubsection{Detailed Description}
The \doxylink{classSettingsWidget}{Settings\+Widget} class provides a customizable settings interface. 

This widget allows users to toggle between a collapsed and expanded view, change themes, and switch units. It interacts with an {\ttfamily \doxylink{classEventManager}{Event\+Manager}} to handle user actions and updates.
