\doxysubsection{Detailed Description}
Subscribes to the joystick topic, convert it and publishes it into cmd\+\_\+vel topic. 

This class essentialy converts joy\+\_\+msg into twist\+\_\+msg. It also publishes the blinker state based on the joystick buttons state.
